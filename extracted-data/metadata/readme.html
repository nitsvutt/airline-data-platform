BACKGROUND

The data contained in the compressed file has been extracted from the Reporting Carrier On-Time Performance (1987-present) data table of the "On-Time" database from the TranStats data library. The time period is indicated in the name of the compressed file; for example, 2001.csv contains data of the 01st month of the year 20.
